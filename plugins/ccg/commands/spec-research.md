# CCG Spec Research

The user invoked:

```text
/ccg:spec-research $ARGUMENTS
```

Use the installed CCG plugin skill `ccg:spec-research`.

Write research and constraints under `.codex/ccg/specs/<name>/`, then validate the spec lifecycle with `spec_manager.js`.
