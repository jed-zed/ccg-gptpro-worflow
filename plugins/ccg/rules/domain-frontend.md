# Domain Frontend Rule

Use this rule for UI, UX, accessibility, responsive layout, component architecture, and visual regression risk.

- Prefer existing framework and design-system patterns.
- For substantial UI work, ask Gemini for read-only review with `--prompt-template frontend`, then Codex implements and verifies.
- Validate responsive behavior, text fit, keyboard/focus states, and accessibility-critical controls.
- Report outcomes in Chinese.
