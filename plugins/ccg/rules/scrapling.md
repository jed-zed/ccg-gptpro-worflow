# Scrapling Safety Rule

Use this rule only for scraping, crawling, extraction, or web data collection workflows.

- Respect robots, rate limits, and site terms.
- Do not scrape credentials, private data, or authenticated content without explicit authorization.
- Do not bypass paywalls, CAPTCHAs, access controls, or bot defenses.
- Do not reuse hidden browser credentials.
- Prefer official APIs and documented export mechanisms.
- If these boundaries make parity unsafe, mark the original behavior `not-copied` with a Codex-native replacement.
