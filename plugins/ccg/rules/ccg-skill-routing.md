# CCG Skill Routing Rule

When a task clearly belongs to a technical domain, load the minimum relevant project documentation before coding.

Useful routing categories:

- Security: pentest, red team, blue team, code audit, vulnerability research, threat intelligence.
- Architecture: API design, caching, cloud native systems, message queues, security architecture.
- AI and MLOps: RAG, agents, LLM security, prompt design, evaluation.
- DevOps: Git workflow, testing, database design, performance, observability, DevSecOps, cost optimization.
- Development languages: Python, Go, Rust, TypeScript or JavaScript, Java or Kotlin, C or C++, Shell.
- Frontend design: UI aesthetics, UX, component patterns, state management, frontend engineering.

When multiple domains match, read only the smallest set of references that directly affects the current work.
