# gen-docs

文档生成器。自动分析模块结构，生成 README.md 和 DESIGN.md 骨架。当用户提到生成文档、创建README、创建DESIGN、文档骨架、文档模板时使用。在新建模块开始时自动触发。

## 执行

执行以下命令：

```bash
node "true/skills/ccg/run_skill.js" gen-docs $ARGUMENTS
```

如需了解此技能的详细说明，请读取: true/skills/ccg/tools/gen-docs/SKILL.md
