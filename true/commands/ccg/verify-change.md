# verify-change

变更校验关卡。分析代码变更，检测文档同步状态，评估变更影响范围。当用户提到变更检查、文档同步、代码审查、提交前检查、diff分析时使用。在设计级变更、重构完成时自动触发。

## 执行

执行以下命令：

```bash
node "true/skills/ccg/run_skill.js" verify-change $ARGUMENTS
```

如需了解此技能的详细说明，请读取: true/skills/ccg/tools/verify-change/SKILL.md
