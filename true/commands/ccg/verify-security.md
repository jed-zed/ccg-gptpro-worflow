# verify-security

安全校验关卡。自动扫描代码安全漏洞，检测危险模式，确保安全决策有文档记录。当用户提到安全扫描、漏洞检测、安全审计、代码安全、OWASP、注入检测、敏感信息泄露时使用。在新建模块、安全相关变更、攻防任务、重构完成时自动触发。

## 执行

执行以下命令：

```bash
node "true/skills/ccg/run_skill.js" verify-security $ARGUMENTS
```
