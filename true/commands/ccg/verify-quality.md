# verify-quality

代码质量校验关卡。检测复杂度、重复代码、命名规范、函数长度等质量指标。当用户提到代码质量、复杂度检查、代码异味、重构建议、lint检查、代码规范时使用。在复杂模块、重构完成时自动触发。

## 执行

执行以下命令：

```bash
node "true/skills/ccg/run_skill.js" verify-quality $ARGUMENTS
```

如需了解此技能的详细说明，请读取: true/skills/ccg/tools/verify-quality/SKILL.md
